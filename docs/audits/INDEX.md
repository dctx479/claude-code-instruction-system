## 使用约定

- **生成时机**：每季度知识资产审计（参见 `memory/best-practices.md` BP-027）、跨环境同步前、重大架构调整前
- **报告生命周期**：归档不删除。后续可在同主题新报告中引用历史版本做趋势分析
- **不在仓库根目录长期保留**：临时报告完成后立即归档到此处
- **关联条目**：报告中已落地为持久规则的部分应在 `memory/best-practices.md` 创建对应 BP 条目，并在此索引交叉引用
